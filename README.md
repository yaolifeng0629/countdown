# Holiday Countdown

一个漂亮简洁的假期倒计时

![](./image.png)

## 特点

-   带漂亮倒计时卡的单页应用程序
-   每个假期的互动动画和效果
-   🎨使用Radix UI进行现代且响应式的设计
-   📅智能排序-最近的假期先出现
-   特殊假期特定动画和图标

## 开始构建

```bash
# Install dependencies
pnpm install
# Run development server
pnpm dev
# Build for production
pnpm build
# Start production server
pnpm start
```
